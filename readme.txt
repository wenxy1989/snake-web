抓奴隶
1.自由人抓奴隶，升级为奴隶主
2.奴隶抓奴隶成为自由人，但需要向自己原来的奴隶主效力
3.奴隶主下的自由人数量超过10，并且自由人职业满足一定条件，奴隶主就成为了管理者（镇长、局长、工厂主）
4.每个自由人藉由其控制的下级自由人的职业达到一定条件，可以转变或者升级为其他角色，可以同时存在
5.属性计算中，相同纬度的属性可以计算，不同纬度的属性计算应该在物体自身之间计算，属性的相互作用有损耗
6.将计算算法写入数据库中，当使用时先获取参与计算的数据，再放入算法中，最后更新数据，将更新后的数据放入数据库中
将技能、属性定义都已数据的形式存储在数据库中，可以使的系统多样化
7.天假令牌机制根据令牌来设定权限
8.单位分为有自主意识和无自主意识两种
